1 Introduction

1.1 Purpose of This Document

The goal of this document is to have a place where you can look for examples of code and to share code with one another. Data analysis is a collaborative process where we all should be actively learning from each other. Think of this document as a log of all of the topics we cover in class and in lab sessions. I will make updates throughout the semester as we cover more advanced material. You can also take this opportunity to either become familiar or to practice using Git and GitHub, a common way for researchers to collaborate.

I encourage you to comment and ask questions on this document and to post any interesting code you come across during the semester. I’ll check this page every Monday Morning before the lab and address any questions during the workshop that afternoon. I hope this proves to be a helpful and beneficial resource, but this document is no substitute for rigorous study of causal inference and R programming.

Let me know if any of this code doesn’t run correctly or if you have any questions or issues!
